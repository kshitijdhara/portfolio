Twitter Social Graph Analysis Project

  • Tech Stack: Apache Spark, Scala, Azure HDInsight, Azure Databricks

Conducted an in-depth analysis of the Twitter social graph, leveraging the power of Apache Spark, a distributed computing framework, to process and analyze large-scale data sets efficiently. Implemented the project using Scala, a functional programming language well-suited for data processing and analysis tasks.

Implemented the PageRank algorithm, a widely used algorithm for ranking web pages, to identify influential users within the Twitter social network. Additionally, developed a recommendation system to suggest potential connections and content based on user interests and social interactions.

Provisioned and configured a Spark cluster on Azure HDInsight, a fully managed, open-source analytics service in the Microsoft cloud. This allowed for seamless scaling and parallel processing of the Twitter data, enabling efficient computation and analysis of the social graph.

Utilized Azure Databricks, a Apache Spark-based analytics platform, to develop, deploy, and manage the Spark applications. Databricks provided a collaborative environment for data scientists and engineers, facilitating code development, experimentation, and collaboration.

Implemented data ingestion pipelines to extract and process Twitter data, including user profiles, social connections, and tweet content. Employed advanced data processing techniques, such as data cleaning, transformation, and feature engineering, to prepare the data for analysis.

Throughout the project, adhered to best practices for distributed computing, data processing, and social network analysis. Documented the findings and insights gained from the analysis, providing valuable recommendations for understanding user behavior, identifying influential users, and improving content recommendations within the Twitter platform.
