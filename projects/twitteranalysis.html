Cloud Computing Team Project

  • Tech Stack: Python, Java, Go, Rust, HTML, CSS, Bootstrap, Docker, Kubernetes, AWS (EC2, EKS, S3), Azure (HDInsight), GCP (Pub/Sub, Dataflow, Data Studio, GAE, BigQuery), MySQL, Apache Spark, PySpark, Bash scripts, Terraform, Git, GitHub, GitHub Actions

Developed a high-performance web service on the cloud for data retrieval and analysis of Twitter data, comprising three microservices:

  1. Blockchain Service: A web service for blockchain validation, implemented in Go.
  2. QR Code Service: A web service for QR code generation and authentication, implemented in Rust, Java, and Python across different iterations.
  3. Twitter Service: A web service for analyzing Twitter data, implemented in Java with a web tier for handling HTTP requests and a storage tier with a MySQL database.

Key tasks included:

  1. Building the web tier of microservices to accept RESTful requests and send responses, deployed on a self-managed Kubernetes cluster on AWS EC2.
  2. Performing ETL (Extract, Transform, Load) on a Twitter dataset (>1TB) from cloud storage, transforming data, and loading into a MySQL database using PySpark on AWS, Azure, or GCP.
  3. Optimizing web frameworks, database schema, and infrastructure for high throughput and low cost.
  4. Testing the scalability of the web application by generating 10,000 user requests using Apache Beam.
  5. Implementing CI/CD methodology using GitHub Actions for streamlined build and deployment.

The project demonstrated skills in building performant and reliable web services on the cloud, handling intensive web traffic, processing large datasets efficiently, optimizing databases, monitoring system performance, and implementing CI/CD practices.[1]
