Social Networking Website (RedditBook) Project

  • Tech Stack: Java, MySQL, Neo4j, MongoDB, Azure (Cloud Shell, Database for MySQL, ARM, Terraform)

Developed a comprehensive social networking website, RedditBook, featuring user timelines and a range of interactive features. Implemented the backend using Java servlets, ensuring efficient handling of user requests and data processing.

Leveraged the power of heterogeneous databases to cater to diverse data storage and retrieval requirements. Deployed a relational database (MySQL) to store structured user data, a graph database (Neo4j) to manage complex relationships and social connections, and a NoSQL document database (MongoDB) to handle unstructured data and user-generated content.

Utilized Azure Cloud Shell, a browser-based command-line experience, to streamline the development and deployment processes. Employed Azure Resource Manager (ARM) templates and Terraform for infrastructure as code (IaC) to provision and manage Azure resources, ensuring consistent and repeatable deployments across different environments.

Implemented robust security measures, such as user authentication and authorization, to safeguard user data and ensure privacy. Incorporated real-time updates and notifications to keep users informed about activities within their social circles.

Designed and implemented intuitive user interfaces using modern web technologies, providing a seamless and engaging experience for users to create and share content, connect with friends, and explore personalized recommendations.

Throughout the project, adhered to industry best practices for web application development, database design, and cloud deployment, ensuring scalability, performance, and maintainability of the social networking platform.
